## Integration Points

### 1. Standalone Use
```typescript
import { useSAM2Detection } from '@/hooks';

const { segmentWithPoints } = useSAM2Detection();
const mask = await segmentWithPoints(video, points);
```

### 2. Combined with Pose Detection
```typescript
// Get wrist position from pose detection
const wrist = poses[0].keypoints[10];

// Segment racket at wrist location
const racketMask = await segmentWithPoints(video, [
  { x: wrist.x, y: wrist.y, label: 1 }
]);
```

### 3. Combined with Object Detection
```typescript
// Detect approximate location with YOLO
const racket = objects.find(o => o.class === 'tennis racket');

// Refine with SAM 2
const preciseMask = await segmentWithBox(video, racket.bbox);
```

### 4. Video Tracking Pipeline
```typescript
// Frame 0: Initial segmentation
const mask0 = await segmentWithPoints(video, initialPoints);

// Frame 1-N: Track using previous mask center
for (let frame of frames) {
  const center = calculateMaskStats(previousMask).center;
  const newMask = await segmentWithPoints(frame, [center]);
  previousMask = newMask;
}
```
